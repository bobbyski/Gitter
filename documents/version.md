# version

Print the current Gitter version.

## Usage

```
python main.py version
```

## Options

None.

## Output

Prints the version string from `pyproject.toml` to stdout.

```
Version 0.1.0
```

## Example

```bash
python main.py version
```

## Notes

- The version is read from the `[project] version` field in `pyproject.toml`.
